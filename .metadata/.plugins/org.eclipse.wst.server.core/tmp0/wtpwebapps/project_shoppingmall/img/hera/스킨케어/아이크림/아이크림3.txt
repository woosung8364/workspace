더 생기있는 눈가로 케어하는 피토-플라센타
눈가 피부에 영양을 공급하는 피토-플라센타 성분이 한결 더 생기있는 눈가로 케어해 줍니다.

활력있는 눈가를 선사하는 카페인
카페인 성분이 피부 스트레스로부터 보호하여 활력있는 눈가를 선사합니다.

슈크림처럼 부드러운 멜팅 밤 텍스쳐
슈크림처럼 부드럽게 감싸주는 멜팅 밤 텍스쳐로, 눈가 둘레의 O존과 관자놀이를 마사지하듯 케어할 수 있습니다.
